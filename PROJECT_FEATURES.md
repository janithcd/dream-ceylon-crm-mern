# Project Features

## Business Workflow

- Inquiry capture and lifecycle management
- Customer history aggregation
- Follow-up scheduling and completion
- Quotation preparation and PDFs
- Quotation-to-booking conversion
- Booking calendar
- Payment tracking and balance calculation
- Financial reporting and CSV export

## Document Generation

- AI-assisted itinerary PDF
- Quotation PDF
- Booking invoice
- Booking receipt
- Payment receipt
- Dynamic company identity
- Configurable watermark, terms and payment instructions

## Administration

- Five administrator roles
- Additional custom permissions
- Active and inactive accounts
- Password reset
- Self-disable and self-delete protection
- Final Super Admin protection
- Permission-aware sidebar, routes and buttons

## Analytics and Monitoring

- Dashboard statistics
- Revenue and booking charts
- Follow-up alerts
- Recent activity
- Searchable activity history
- Success and failed action records

## Website Readiness

- Public destinations API
- Public packages API
- Public vehicle API
- Public homepage API
- Public inquiry submission API
- Foundation for a Next.js website
